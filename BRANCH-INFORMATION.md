# Branch Information

**Branch name:** `writing`
**Objective:** Develop stories from raw idea to written chapters — through structured world-building, design, and cooperative writing.
**Workflow path:** `workflow/stages/`
**Agent file:** `AGENTS.md`

## What this branch is for

Use this branch when writing long-form fiction:
- Web novels (serialized, episodic)
- Traditional novels or novellas
- Manga / graphic novel scripts
- Screenplays and audio dramas
- Game narratives
- Any other story-driven long-form format

Each story is its own isolated entity within the project. Multiple stories can coexist.

## How to use

Clone this branch only:

```bash
git clone --branch writing --single-branch <repo-url> my-story
cd my-story
claude  # or: gemini, etc.
/start-stage 1-1
```

## Cross-branch decisions

Changes that affect all workflow branches are documented in the `main` branch at `docs/cross-branch-decisions.md`.
